FFlag quick start

Everything you need to run fflag.ahk — no extra files required.

Before you start

  • Install AutoHotkey v2 from autohotkey.com
  • Download fflag.ahk from this site
  • Create an account on the Register page (login inside the tool)
  • Internet connection (offsets load from GitHub)

Steps

  1. Run as admin

    Right-click fflag.ahk → Run as administrator. The script will ask for elevation if you double-click it.

  2. Log in

    Use your Leviathan account. You stay signed in for 30 days, then log in again.

  3. Join Roblox

    Launch a game with the normal client, Bloxstrap, or Fishstrap.

  4. Press Attach

    The tool reads your running RobloxPlayerBeta.exe, picks the matching offset version, and loads flags from GitHub automatically.

  5. Add flags & Apply

    Add or import FFlags, then click Apply to Roblox. Use Save Flags to keep your list.

Settings (gear icon)

  • Version search — type part of a version hash (e.g. 5cf227) to filter the list
  • Click a version — loads offsets for that branch manually
  • Auto — uses the running client when Attach is used
  • Reload — refreshes the version list from GitHub + local installs

Portable / RAR Roblox

If Roblox runs from a temp folder (no version-… in the path), Attach uses the exe build number and our build_map.json on GitHub to find the right offsets.
